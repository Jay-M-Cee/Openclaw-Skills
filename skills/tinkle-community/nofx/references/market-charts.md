# NOFX Market 页面使用

## 访问方式

URL: `https://nofxai.com/market`

## 功能概览

### K线图表

- TradingView 风格 K 线图
- 支持多时间周期: 1m, 5m, 15m, 30m, 1h, 4h, 1d
- 实时价格更新

### 技术指标

| 指标 | 说明 | 用法 |
|------|------|------|
| EMA | 指数移动平均 | 趋势方向 |
| MACD | 异同移动平均 | 动量/背离 |
| RSI | 相对强弱指数 | 超买超卖 |
| BOLL | 布林带 | 支撑阻力/波动率 |
| Volume | 成交量 | 量价配合 |

### 订单簿深度

- 买卖挂单分布
- 大单标记
- 买卖压力对比

### 实时数据

- 当前价格
- 24h 涨跌幅
- 24h 成交量
- 资金费率
- 持仓量

## 浏览器自动化

### 打开 Market 页面

```javascript
browser.navigate("https://nofxai.com/market")
browser.snapshot()
```

### 切换交易对

```javascript
// 点击交易对选择器
// 输入或选择目标交易对
browser.act({kind: "click", ref: "symbolSelector"})
browser.act({kind: "type", ref: "searchBox", text: "ETHUSDT"})
```

### 切换时间周期

```javascript
// 点击对应时间按钮
browser.act({kind: "click", ref: "timeframe1h"})
```

## 数据解读

### K线形态

| 形态 | 含义 | 操作建议 |
|------|------|----------|
| 大阳线 | 强势上涨 | 顺势做多 |
| 大阴线 | 强势下跌 | 顺势做空 |
| 十字星 | 犹豫/转折 | 等待确认 |
| 锤子线 | 底部反转 | 考虑做多 |
| 倒锤子 | 顶部反转 | 考虑做空 |

### RSI 信号

| 数值 | 状态 | 操作 |
|------|------|------|
| > 70 | 超买 | 考虑做空/止盈 |
| < 30 | 超卖 | 考虑做多/抄底 |
| 50 附近 | 中性 | 等待方向 |

### MACD 信号

| 信号 | 含义 |
|------|------|
| 金叉 | DIF 上穿 DEA，看涨 |
| 死叉 | DIF 下穿 DEA，看跌 |
| 顶背离 | 价格新高 MACD 不新高，警惕回调 |
| 底背离 | 价格新低 MACD 不新低，可能反弹 |

### 布林带用法

| 位置 | 含义 |
|------|------|
| 触及上轨 | 超买，可能回调 |
| 触及下轨 | 超卖，可能反弹 |
| 收窄 | 即将突破 |
| 扩张 | 波动加大 |
